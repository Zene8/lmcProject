LMC Instruction Set Quick Reference

Input/Output

Instruction Opcode Description
INP 901 Reads a number from the input and stores it in the accumulator.
OUT 902 Writes the value from the accumulator to the output.

Memory Operations

Instruction Opcode Description
LDA 5xx Loads the value from the specified memory address into the accumulator.
STA 3xx Stores the value from the accumulator to the specified memory address.

Arithmetic

Instruction Opcode Description
ADD 1xx Adds the value from the specified memory address to the accumulator.
SUB 2xx Subtracts the value from the specified memory address from the accumulator.

Control Flow

Instruction Opcode Description
BRA 6xx Branches to the specified memory address unconditionally.
BRZ 7xx Branches to the specified memory address if the accumulator is zero.
BRP 8xx Branches to the specified memory address if the accumulator is positive or zero.
HLT 000 Halts the program.

Data

Instruction Opcode Description
DAT N/A Reserves a memory location and initializes it with a value.
